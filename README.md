# Torrent Extractor

## Requisitos
	- Python3.x
	- Instalar dependências

## Para executar
- `pip3 install -r requirements.txt`
- `python main.py`

## Sobre o app
	Evite os anúncios e redirects dos sites de torrents, cole o link do post com o conteúdo que deseja baixar e tenha acesso direto aos magnet links.
